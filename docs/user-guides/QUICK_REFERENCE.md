# 📋 Шпаргалка по приёмке товара

## 🔵 СУПЕРВИЗОР - Быстрая инструкция

```
┌─────────────────────────────────────────────────┐
│  ШАГ 1: ПОДТВЕРДИТЬ ПРИХОД                     │
├─────────────────────────────────────────────────┤
│  1. Открыть "Приходы"                          │
│  2. Кликнуть на приход (статус DRAFT)          │
│  3. Нажать "Подтвердить"                       │
│  ✅ Статус → CONFIRMED                          │
└─────────────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────────────┐
│  ШАГ 2: НАЧАТЬ ПРИЁМКУ                         │
├─────────────────────────────────────────────────┤
│  1. Кликнуть на приход (статус CONFIRMED)      │
│  2. Нажать "Начать приёмку"                    │
│  ✅ Статус → RECEIVING                          │
│  ✅ Созданы задания RECEIVING                   │
└─────────────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────────────┐
│  ОПЕРАТОРЫ РАБОТАЮТ                             │
│  (Приёмка товара, сканирование)                │
└─────────────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────────────┐
│  ШАГ 3: ЗАВЕРШИТЬ ПРИЁМКУ                      │
├─────────────────────────────────────────────────┤
│  1. Кликнуть на приход (статус RECEIVING)      │
│  2. Нажать "Завершить приёмку"                 │
│  ✅ Статус → RECEIVED                           │
└─────────────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────────────┐
│  ШАГ 4: НАЧАТЬ РАЗМЕЩЕНИЕ                      │
├─────────────────────────────────────────────────┤
│  1. Кликнуть на приход (статус RECEIVED)       │
│  2. Нажать "Начать размещение"                 │
│  ✅ Статус → PLACING                            │
│  ✅ Созданы задания PLACEMENT                   │
└─────────────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────────────┐
│  ОПЕРАТОРЫ РАБОТАЮТ                             │
│  (Размещение паллет в ячейки)                  │
└─────────────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────────────┐
│  ШАГ 5: ПРИНЯТЬ ПРИХОД                         │
├─────────────────────────────────────────────────┤
│  1. Кликнуть на приход (статус PLACING)        │
│  2. Нажать "Принять"                           │
│  ✅ Статус → STOCKED                            │
│  ✅ ПРИХОД ЗАВЕРШЁН! 🎉                         │
└─────────────────────────────────────────────────┘
```

## 🟢 ОПЕРАТОР - Быстрая инструкция

### Приёмка товара (RECEIVING)

```
┌─────────────────────────────────────────────────┐
│  ШАГ 1: НАЙТИ ЗАДАНИЕ                          │
├─────────────────────────────────────────────────┤
│  1. Открыть "Терминал"                         │
│  2. Вкладка "Мои задачи" или "Все задачи"     │
│  3. Двойной клик на задание RECEIVING          │
└─────────────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────────────┐
│  ШАГ 2: НАЗНАЧИТЬ И НАЧАТЬ                     │
├─────────────────────────────────────────────────┤
│  1. Нажать "Назначить" → выбрать себя          │
│  2. Нажать "Начать"                            │
│  ✅ Статус → В РАБОТЕ                           │
└─────────────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────────────┐
│  ШАГ 3: СКАНИРОВАТЬ (вкладка "Факт")          │
├─────────────────────────────────────────────────┤
│  1. 📷 Паллета: PLT-001                        │
│  2. 📷 Баркод: 1234567890                      │
│  3. ✏️ Количество: 100                         │
│  4. ☑️ Брак (если есть)                        │
│  5. Нажать "Отправить"                         │
│  🔄 Повторить для всех паллет                   │
└─────────────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────────────┐
│  ШАГ 4: ЗАВЕРШИТЬ                              │
├─────────────────────────────────────────────────┤
│  1. Нажать "Завершить"                         │
│  2. Подтвердить (если есть расхождения)        │
│  ✅ Задание ВЫПОЛНЕНО! ✅                        │
└─────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────────────┐

└─────────────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────────────┐

│  1. Вкладка "Документ" → смотрим целевую ячейку │
│     🎯 Целевая ячейка: A-01-05                  │
│  2. Вкладка "Факт":                            │
│     - 📷 Паллета: PLT-001                      │
│     - 📷 Баркод: 1234567890                    │
│     - ✏️ Количество: 100                       │
│     - 📍 Ячейка: A-01-05  ← ОБЯЗАТЕЛЬНО!       │
│  3. Нажать "Отправить"                         │
│  4. Нажать "Завершить"                         │
│  ✅ Товар РАЗМЕЩЁН! ✅                           │
└─────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────────────┐
│  МОНИТОРИНГ ПРИХОДОВ                            │
├─────────────────────────────────────────────────┤
│  "Приходы" → смотреть статусы                  │
│                                                 │
│  🔴 DRAFT     → Сообщить супервизору           │

                    ↓
┌─────────────────────────────────────────────────┐
│  МОНИТОРИНГ ЗАДАНИЙ                             │
├─────────────────────────────────────────────────┤
│  "Задания" → фильтр по receiptId               │
│                                                 │
│  📊 Считать:                                   │
│     - Всего заданий                            │
│     - Выполнено (COMPLETED)                    │
│     - В работе (IN_PROGRESS)                   │
│     - Не начато (NEW/ASSIGNED)                 │
│                                                 │
│  ⚠️ Если задание висит > 30 мин                │
│     → Сообщить супервизору                     │
└─────────────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────────────┐
│  ПРОВЕРКА ОСТАТКОВ                              │
├─────────────────────────────────────────────────┤
│  "Остатки" → фильтр "ID прихода"               │
│                                                 │
│  ✅ Проверить:                                 │
│     - Количество совпадает с ожидаемым?        │
│     - Все паллеты размещены?                   │
│     - Есть ли брак?                            │
└─────────────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────────────┐
│  АНАЛИТИКА                                      │
├─────────────────────────────────────────────────┤
│  "Аналитика" → выбрать период → "Обновить"    │
│                                                 │
│  📈 Отслеживать:                               │
│     - % расхождений (норма < 5%)               │
│     - % брака (норма < 2%)                     │
│     - Среднее время приёмки                    │

```

## 🚨 Типичные проблемы и решения

| Проблема | Решение |
|----------|---------|
| Кнопка серая/неактивна | Проверить статус, выделить строку, обновить список |
| Не могу отсканировать | Ввести код вручную, проверить сканер |
| "Задание зависло" | PC_Operator → сообщить супервизору + имя оператора |
| Расхождение количества | Подтвердить реальное количество, супервизор разберётся позже |
| Забыл завершить задание | Терминал → Мои задачи → открыть → Завершить |
| Товар повреждён | Отметить галочку "Брак", заполнить описание |
| Не вижу задания в "Мои задачи" | Проверить "Все задачи", возможно не назначено |

## 📞 Кому звонить

| Вопрос | Кто поможет |
|--------|-------------|
| Приход не подтверждён | **Supervisor** |
| Задание не назначено | **Supervisor** |
| Задание "зависло" | **Supervisor** |
| Нужна помощь со сканированием | **Другой Operator** |
| Сломался сканер | **Техподдержка** |
| Приложение не работает | **Admin** |
| Непонятно что делать | **Supervisor** |

## ✅ Быстрый чек перед уходом домой

### Супервизор:
- [ ] Все новые приходы обработаны (нет DRAFT)
- [ ] Нет приходов, застрявших в RECEIVING/PLACING
- [ ] Все задания либо COMPLETED, либо назначены

### Оператор:
- [ ] Все мои задания завершены (нет IN_PROGRESS)
- [ ] Сканер выключен и поставлен на зарядку
- [ ] Рабочее место убрано

### PC_Operator:
- [ ] Супервизор проинформирован о всех проблемах дня
- [ ] Сформирован отчёт по расхождениям/браку (если есть)
- [ ] Завтрашние приходы проверены (если есть DRAFT на завтра)
