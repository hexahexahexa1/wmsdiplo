# ФИНАЛЬНОЕ ИСПРАВЛЕНИЕ: Ширина кнопок в панели фильтров

## Проблема
Текст "Очистить" обрезается до "Очи..." на кнопке из-за недостаточной ширины (120px).

## Финальное решение

### Оптимальные ширины кнопок:

**Кнопка "Поиск"**:
- Ширина: **110px**
- Высота: 40px

**Кнопка "Очистить"**:
- Ширина: **140px** ✅
- Высота: 40px

**Кнопка "Закрыть"** (в диалоге истории):
- Ширина: **150px** ✅
- Высота: 35px

### Структура интерфейса:

```
Строка 1: [Код SKU] [Код ячейки] [Баркод паллеты] [ID прихода] [Статус]
                     (все по 150px)

Строка 2: [Поиск: 110px] [Очистить: 140px]
```

## Применение

Перезапустите desktop-client:

```bash
gradle :desktop-client:run
```

## Результат

✅ **"Поиск"** - полностью виден (110px достаточно)  
✅ **"Очистить"** - полностью виден (140px достаточно)  
✅ **"Закрыть"** - полностью виден (150px достаточно)  
✅ Кнопки выглядят пропорционально  
✅ Интерфейс чистый и читаемый

**Примечание**: Если текст все еще обрезается, возможно используется специфичный шрифт или масштабирование. В этом случае можно дополнительно увеличить ширину кнопки "Очистить" до 150px или использовать более короткий текст "Сброс".
