# Исправление: Текст на кнопках обрезается

## Проблема
Текст на кнопках "Очистить" и "Закрыть" не помещается целиком из-за недостаточной ширины кнопок.

## Решение
Увеличена ширина кнопок для корректного отображения русского текста:

### Изменения в файле:
`desktop-client/src/main/java/com/wmsdipl/desktop/DesktopClientApplication.java`

**Кнопка "Поиск"**:
- Было: 120px
- Стало: 100px (оптимизировано)

**Кнопка "Очистить"**:
- Было: 120px
- Стало: 150px ✅

**Кнопка "Закрыть"** (в диалоге истории):
- Было: 120px
- Стало: 150px ✅
- Добавлена высота: 35px

## Применение изменений

Перезапустите desktop-client:

```bash
# Если запущен - остановите (закройте окно или Ctrl+C)
# Запустите заново
gradle :desktop-client:run
```

## Результат
✅ Текст "Очистить" полностью помещается на кнопке  
✅ Текст "Закрыть" полностью помещается на кнопке  
✅ Кнопки выглядят пропорционально

Код скомпилирован успешно, готов к запуску.
