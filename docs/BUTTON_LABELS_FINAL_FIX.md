# ОКОНЧАТЕЛЬНОЕ РЕШЕНИЕ: Короткие названия кнопок

## Проблема
При использовании длинных названий кнопок ("Поиск", "Очистить") текст обрезается из-за padding и font-size в CSS стиле `refresh-btn`, даже при увеличении ширины кнопок до 140-150px.

## Решение: Короткие, понятные названия

### Изменения текста кнопок:

| Было | Стало | Ширина |
|------|-------|--------|
| "Поиск" → "По..." | **"Найти"** | 100px |
| "Очистить" → "Очист..." | **"Сброс"** | 100px |

### Преимущества:

✅ **"Найти"** - короткое, интуитивно понятное  
✅ **"Сброс"** - короткое, стандартное название для очистки фильтров  
✅ Обе кнопки имеют одинаковую ширину (100px) - визуально гармонично  
✅ Текст полностью помещается без обрезания  
✅ Смысл кнопок остается абсолютно понятным

### Альтернативные варианты (если понадобится):

- "Искать" / "Очистить"
- "Найти" / "Очистить" (если увеличить кнопку до 150px)
- "OK" / "Сброс"
- "Применить" / "Сброс"

## Структура интерфейса:

```
Строка 1: [Код SKU] [Код ячейки] [Баркод паллеты] [ID прихода] [Статус]
                     (все по 150px)

Строка 2: [Найти: 100px] [Сброс: 100px]
```

## Применение

Перезапустите desktop-client:

```bash
gradle :desktop-client:run
```

## Файл изменен

`desktop-client/src/main/java/com/wmsdipl/desktop/DesktopClientApplication.java`

**Строки 557-565**: Изменены названия и ширина кнопок

## Результат

✅ Текст на обеих кнопках полностью виден  
✅ Кнопки одинаковой ширины - выглядят аккуратно  
✅ Названия короткие, но интуитивно понятные  
✅ Консистентность с общим дизайном приложения

Код успешно скомпилирован и готов к использованию!

---

**Примечание**: Если в будущем потребуется вернуть длинные названия, можно:
1. Увеличить ширину кнопок до 150-160px
2. Уменьшить font-size в CSS стиле `refresh-btn`
3. Уменьшить padding в CSS стиле `refresh-btn`
